
## Environment setup 

- create .env file
- set database name (mysql)
- php command 
```bash
php artisan serve
```

## Task-1  Blog Post CRUD API 
1. For Create a Post 

```bash
http://127.0.0.1:8000/api/create-post

```
2. For List All Posts

```bash
http://127.0.0.1:8000/api/posts

```
3. For View a Single Post

```bash
http://127.0.0.1:8000/api/posts/1

```
